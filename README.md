## Métodos

* Metodo principal:
    ````java
    public void tocar_playlist()
    ````

* Outros métodos:
    - Criando a playlist
        ```java
        public void playlist()
        ````
    - Criando musicas e filmes
        ````java
    	private void criar_musicas()
        private void criar_filmes()
        ````
    - Para ranomizar as midias
        ```java
        private void Midia[] random()
        ````
